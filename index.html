Gmail Cleanup

Setup

This tool runs entirely in your browser. No data is sent to any server. You need a Google Cloud project with the Gmail API enabled.

  1. Go to Google Cloud Console > Credentials
  2. Create a project (or use an existing one)
  3. Enable the Gmail API under "APIs & Services"
  4. Create an OAuth 2.0 Client ID (type: Web application)
  5. Add http://localhost (or wherever you serve this file) to Authorized JavaScript origins
  6. Create an API Key (restrict it to Gmail API for safety)
  7. If the app is in "Testing" mode, add your Gmail address as a test user under the OAuth consent screen

Credentials are stored in localStorage so you don't have to re-enter them. They never leave your browser.
